Services

To access the Services section, login to the web app by clicking the Login button at the top-right corner of the navigation bar (If you do not have an account, click the Register button next to the Login button). To access the Services section, now that you're logged in, click Services, then at dropdown, click Create a model for the 1st window of Services, or click Services, then at dropdown, click Pretrained model for the 2nd window of Services

The services section contains 2 windows: Create new model and Pretrained models.

Create New Model

In this section, users can select an existing dataset or upload their own. After uploading, they can normalize the data, choose parameters, and execute the k-NN algorithm. Once executed, an evaluation appears, displaying metrics per class, average metrics, and the best parameters. To save the model, enter its name and click 'Save model'.

The picture on top, shows the half top of the evaluation window, in which includes the 'Metrics per Class' and the 'Average Metrics' tables.

The picture below the top, is the other half of the evaluation window, in which shows the 'Best parameters' table, the text input for the model's name, and the 'Save model' button.

Pretrained Models

Previously built models appear here. Users can select a pretrained model and apply it to an unclassified dataset (which must match the class and features of the selected model) to predict class values. This process provides both average and per-class metrics, and results can be exported in CSV format.

The picture on top, shows the half top of the classified dataset window, in which includes the 'Metrics per Class' table (to see the table, the show buttom must be clicked) and the info button.

The picture below the top, is the other half of the classified dataset window, in which shows the 'Average Metrics' table (to see the table, the show button must be clicked) and the 'Export to .csv' button.
